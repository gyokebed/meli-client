### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:gyokebed/meli-client.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run dev
   ```sh
   npm run dev
   ```
4. Run build
   ```sh
   npm run build
   ```
